# LED Cube Controller

This project is a proof of concept LED Cube Controller application developed using Python and Tkinter.

## Features
- Mode selection (Static, Blink, Chase, Burst, Off)
- Visual LED matrix simulation
- Continuous animation system
- Reset functionality (restarts active mode)
- Serial communication layer (simulation or hardware)

## Technologies Used
- Python
- Tkinter
- PySerial

## Purpose
This application demonstrates how a user interface, control logic, and communication system can work together to simulate and control LED behavior.

## Future Improvements
- Preset system for saving patterns
- Brightness control
- Animation speed control
- Expanded LED visualization
- Hardware integration with Arduino or ESP32
